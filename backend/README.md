# Tanyarat Shop API

API สำหรับระบบร้านค้าออนไลน์ Tanyarat ที่รองรับฟังก์ชันต่างๆ ดังนี้

## ฟังก์ชันหลัก

### ระบบสมาชิก
- ลงทะเบียน / เข้าสู่ระบบ
- จัดการข้อมูลส่วนตัว
- ประวัติการสั่งซื้อ
- ที่อยู่จัดส่ง

### ระบบร้านค้า
- แสดงรายการสินค้า
- ค้นหาและกรองสินค้า
- ตะกร้าสินค้า
- รายละเอียดสินค้า

### ระบบชำระเงิน
- รองรับหลายช่องทางการชำระเงิน
- บันทึกประวัติการชำระเงิน
- ใบเสร็จและใบกำกับภาษี

### ระบบติดตามการจัดส่ง
- สถานะการจัดส่ง
- การแจ้งเตือนเมื่อสถานะเปลี่ยน
- ข้อมูลการติดตามพัสดุ

### ระบบจัดการสต็อกสินค้า
- ตรวจสอบสต็อกสินค้า
- แจ้งเตือนเมื่อสินค้าใกล้หมด
- ประวัติการเคลื่อนไหวของสินค้า

### ระบบ Admin
- จัดการสินค้า
- จัดการคำสั่งซื้อ
- จัดการสมาชิก
- รายงานและสถิติ
- การตั้งค่าระบบ

### ระบบแจ้งเตือน
- Webhook สำหรับการแจ้งเตือนการสั่งซื้อ
- การแจ้งเตือนผ่านอีเมล

## โครงสร้างโปรเจค

```
api/
├── src/
│   ├── controllers/   # ตัวควบคุมสำหรับจัดการลอจิกของแต่ละเส้นทาง
│   ├── middlewares/   # Middleware สำหรับตรวจสอบการเข้าถึง
│   ├── models/        # โมเดลฐานข้อมูล MongoDB
│   ├── routes/        # เส้นทาง API
│   ├── utils/         # ฟังก์ชันช่วยเหลือต่างๆ
│   └── index.js       # ไฟล์หลักของแอปพลิเคชัน
├── Dockerfile         # สำหรับสร้าง Docker image
└── package.json       # รายการ dependencies
```

## การติดตั้งและการใช้งาน

1. ติดตั้ง dependencies:
   ```
   npm install
   ```

2. รันในโหมดพัฒนา:
   ```
   npm run dev
   ```

3. รันในโหมดการผลิต:
   ```
   npm start
   ```

4. การใช้งานกับ Docker:
   ```
   docker-compose up
   ```
